Add Issue
Steps:
  1. Home Page>Add Icon.
  2. Choose Issue Type.
  3. Choose the Specified Department .
  4. Choose One or More Brands from The Previously selected Department.
  5. Choose a User or more of the pre-selected Brands.
  6. Enter Additional Information Related in Issue
  7. Upload Attachments if You Want
Notes:
  • If you click Reset button all previous information entered will remove
  • If you click Save button will save version of information as draft.
  • If you click Submit button will add new issue and redirection to issue list .
